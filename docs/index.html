BWT — Brain Write Transaction

  1. 01 Intent validation
  2. 02 .bak backup creation
  3. 03 Directory creation
  4. 04 Document .tmp write
  5. 05 Index .tmp update
  6. 06 Manifest .tmp update
  7. 07 Digest .tmp update
  8. 08 Validate integrity
  9. 09 Atomic rename ✓

If any step fails → full automatic rollback

Get Started in 30 Seconds

# Install globally via npm
npm install -g @ernham/brain-cli

# Initialize your Brain directory
brain-cli init
# Recall memories at session start
brain-cli recall -b -g "project keywords"

# Brief mode (-b) for concise output
# Adjust -g (goal) to match your context
# Store a new memory
brain-cli write '{
  "action": "create",
  "sourceRef": "10_projects/my-app/note.md",
  "content": "# My Note\nImportant decision...",
  "record": {
    "scopeType": "project",
    "scopeId": "my-app",
    "type": "note",
    "title": "Architecture decision",
    "summary": "Chose REST over GraphQL",
    "tags": ["domain/api", "intent/decision"],
    "sourceType": "candidate"
  }
}'
# Register thinking strategies
brain-cli meta-seed

# Rate what worked
brain-cli meta-feedback positive

# Auto-improve: boost winners, demote losers
brain-cli meta-learn
