Architecture

BWT — Brain Write Transaction

  1. Step 1 ✓ Intent validation
  2. Step 2 ✓ .bak backup creation
  3. Step 3 ✓ Directory creation
  4. Step 4 ✓ Document .tmp write
  5. Step 5 ✓ Index .tmp update
  6. Step 6 ✓ Manifest .tmp update
  7. Step 7 ✓ Digest .tmp update
  8. Step 8 ✓ Validate integrity
  9. Step 9 ✓ Atomic rename

⚠️ If any step fails → full rollback
